# PRD: Web 管理界面

> **状态**：已实现（Phase 2.1 MVP）
> **模块**：docrank-webui
> **日期**：2025-03-08

---

## 1. 背景与目标

### 1.1 问题陈述

DocRank 的知识库管理（写入/查看/删除文档、测试检索效果）目前只能通过 curl 或代码操作，对非开发者或调试人员极不友好。

### 1.2 目标

提供内嵌于 Spring Boot 的 Web 管理界面，支持搜索测试、文档写入、删除管理和统计查看，无需安装额外服务。

### 1.3 非目标（当前 MVP）

- 不提供文档列表浏览（需 List API 支持）
- 不提供写入速率图表和高频查询统计
- 不实现用户认证

---

## 2. 用户故事

```
作为开发者，
我希望在浏览器中输入查询词并查看检索结果，
以便快速验证知识库内容和检索质量。
```

```
作为运维，
我希望在界面上上传文件写入知识库，
以便不需要写代码或 curl 命令。
```

---

## 3. 功能需求

### 3.1 搜索 Tab

- **P0** 输入框：query / scope（可选）/ topK（3/5/10/20）
- **P0** 显示结果列表：title / content 摘要（4 行）/ 分数 / 语言 / doc_id / tags / section
- **P0** 分数可视化（紫色进度条）
- **P0** 无结果时显示提示

### 3.2 写入 Tab

- **P0** 文本写入：title / content / scope / importance / tags
- **P0** 文件上传：支持 PDF/MD/HTML/DOCX/TXT/JSON，拖拽上传
- **P0** 写入成功后显示 doc_id 和 chunk 数量
- **P0** 写入后自动刷新 Stats

### 3.3 管理 Tab

- **P0** 按 doc_id 删除单个文档
- **P0** 按 scope 批量删除（GDPR 清除），带警告提示
- **P0** 操作结果实时反馈（成功/错误）

### 3.4 统计 Tab

- **P0** 服务健康状态指示灯（绿/红）
- **P0** 向量索引 chunk 总数
- **P0** BM25 索引 chunk 总数
- **P0** 系统信息表（版本/模型/检索策略/语言支持/API 链接）
- **P0** 手动刷新按钮

### 3.5 部署方式

- **P0** 静态文件内嵌于 Spring Boot jar（`src/main/resources/static/docrank-ui/`）
- **P0** 访问地址：`http://localhost:8080/docrank-ui/`
- **P0** 添加 `docrank-webui` 依赖后自动生效（无需额外配置）

---

## 4. 非功能需求

| 维度 | 要求 |
|------|------|
| 技术栈 | 纯前端（Alpine.js + Tailwind CSS），不引入 Node.js 构建步骤 |
| 部署 | 零独立部署，内嵌 jar，随 Spring Boot 启动 |
| 响应式 | 支持桌面浏览器（1024px+） |
